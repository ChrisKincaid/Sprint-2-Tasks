Object Oriented Programming (OOP)

-One of the most popular programming paradigms in the world.
>>A programming paradigm is a style or way of programming.

-There are many different programming paradigms:
>>Functional Programming
>>Procedural Programming
>>Event Driven Programming

-OOP is a style of programming that focuses on objects and data rather than actions and logic.
-A way to manage and think about state.
-Models interactions within and between states.
-Objects have data and code associated with them.

-Four main ideas of OOP:
>>Encapsulation
>>Abstraction
>>Inheritance
>>Polymorphism

-JavaScript Specifics
>>JS is NOT a traditional OOP language.
>>Classes are built on top of prototypes.
>>There is an actual Object type in JS.
>>Most/All OOP concepts can be simulated/implemented in JS.
